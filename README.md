# Wedding Guest List Manager

**Created with AI assistance using Cursor**  

This project is a console-based **Wedding Guest List Manager** written in C++. It allows you to:

- Add and remove guests
- Update RSVP status
- Display all guests and total attendees
- Save and load guest lists from a file

---

## Purpose

This repository was created as a **test of AI prompts and coding abilities using Cursor**, an AI assistant for generating and structuring programs.  

It is personally relevant to me because I am getting married and wanted a fun way to **practice programming while managing a real-life event**.

---

## Features

- Fully object-oriented C++ design (`Guest`, `GuestListManager`, utilities)
- File persistence with `.txt` guest lists
- Input validation for names, party sizes, and RSVP statuses
- Menu-driven console interface

---

## Usage

1. Clone or download the repository.
2. Open `WeddingGuestList.sln` in Visual Studio.
3. Build and run the project.
4. Use the menu to manage your wedding guest list interactively.
